Aprende HTML Moderno

HTML es el lenguaje fundamental de la web. Con él podrás crear la estructura básica de cualquier página web. En este tutorial, aprenderás desde los conceptos básicos hasta las características más modernas de HTML5.

Estructurado

Organiza el contenido de forma lógica

Accesible

Compatible con lectores de pantalla

Responsivo

Se adapta a todos los dispositivos

¿Para qué sirve HTML?

HTML define la estructura y el significado del contenido web. Con él describes encabezados, párrafos, formularios, tablas y cualquier bloque que después podrás enriquecer con CSS y JavaScript.

Web corporativa

Landing pages, portfolios y sitios institucionales

Catálogos online

Estructura fichas de producto y formularios de compra

Documentación

Guías técnicas y recursos educativos accesibles

Paso a paso: Define el objetivo, traza el esqueleto con header/main/footer, añade semántica con article/section y revisa la accesibilidad.

<main>
    <section class="hero" role="banner">
        <h1>Academia DAM</h1>
        <p>Formación práctica en desarrollo multiplataforma.</p>
        <a class="cta" href="#cursos">Ver cursos</a>
    </section>

    <section id="cursos">
        <h2>Oferta formativa</h2>
        <article class="card">
            <h3>Programación</h3>
            <p>Algoritmos, POO y patrones.</p>
        </article>
    </section>
</main>

Estructura Básica

Todo documento HTML tiene una estructura fundamental que define su organización. Aquí tienes un ejemplo de la estructura básica:

DOCTYPE

Define el tipo de documento

HTML

Elemento raíz del documento

HEAD

Metadatos y recursos

BODY

Contenido visible

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Hola, Mundo</h1>
    <p>Este es mi primer documento HTML.</p>
</body>
</html>

¡A Practicar! Crea tu propio archivo ejercicio.html y replica la estructura básica. Cambia el título y el contenido del párrafo por algo personal.

HTML Semántico

El HTML semántico da significado a la estructura de tu página, mejorando la accesibilidad y el SEO.

Nav

Menú de navegación

Article

Contenido independiente

Section

Secciones temáticas

Footer

Pie de página

<header>
    <nav>
        <ul>
            <li><a href="#inicio">Inicio</a></li>
            <li><a href="#servicios">Servicios</a></li>
        </ul>
    </nav>

<main>
    <article>
        <h1>Título del Artículo</h1>
        <section>
            <h2>Sección 1</h2>
            <p>Contenido de la sección...</p>
        </section>
    </article>
    
    <aside>
        <h3>Contenido Relacionado</h3>
        <p>Enlaces y recursos adicionales...</p>
    </aside>
</main>

<footer>
    <p>© 2025 Mi Sitio Web</p>
</footer>

Formato de Texto Avanzado

HTML ofrece una gran variedad de etiquetas para dar formato específico al texto, más allá de la negrita y la cursiva.

<p>Esto es <strong>importante</strong> 
y esto es <em>enfatizado</em>.</p>

<p>H<sub>2</sub>O y 2<sup>3</sup> = 8</p>

<p><mark>Texto resaltado</mark> para destacar.</p>

¡A Practicar! Añade un nuevo párrafo que contenga al menos una palabra en negrita, otra en cursiva, y una fórmula química usando sub o sup.

Metaetiquetas y SEO

Las metaetiquetas en el head proporcionan información crucial para los navegadores y los motores de búsqueda.

<head>
    <!-- Título (muy importante para SEO) -->
    <title>Título Descriptivo de la Página</title>

    <!-- Descripción para buscadores -->
    <meta name="description" 
          content="Una descripción concisa de tu página.">

    <!-- Open Graph para redes sociales -->
    <meta property="og:title" 
          content="Título para Redes Sociales">
    <meta property="og:description" 
          content="Descripción al compartir.">
    <meta property="og:image" 
          content="https://ejemplo.com/imagen.jpg">
</head>

Atributos Avanzados

Además de los atributos comunes, HTML ofrece otros para funcionalidades más específicas.

Atributos data-*

Permiten almacenar información extra y personalizada en cualquier elemento HTML, que luego puede ser fácilmente accedida con JavaScript.

<div id="usuario" data-id="12345" data-role="admin">
    John Doe
</div>

<script>
const usuarioDiv = document.getElementById('usuario');
console.log(usuarioDiv.dataset.id); // "12345"
console.log(usuarioDiv.dataset.role); // "admin"
</script>

Gráficos en la Web

HTML5 introdujo dos potentes formas de crear gráficos directamente en el navegador.

Canvas

Un lienzo para dibujar gráficos 2D y 3D dinámicamente con JavaScript

SVG

Gráficos vectoriales escalables sin perder calidad

<!-- Un círculo simple con SVG -->
<svg width="100" height="100">
   <circle cx="50" cy="50" r="40" 
           stroke="black" stroke-width="3" fill="red" />
</svg>

<!-- Un lienzo para dibujar con JavaScript -->
<canvas id="miCanvas" width="200" height="100" 
        style="border:1px solid #000;"></canvas>

¡A Practicar! Copia el código del círculo SVG en tu página. Intenta cambiar el color de relleno a "blue" y el radio a "20".

APIs de HTML5

HTML5 no es solo etiquetas, también es un conjunto de APIs de JavaScript que dan superpoderes al navegador.

Web Storage

Almacena datos en el navegador con localStorage y sessionStorage

Geolocation

Permite al usuario compartir su ubicación

Web Workers

Ejecuta scripts en segundo plano sin congelar la UI

// Ejemplo de uso de Local Storage
localStorage.setItem('nombreUsuario', 'Ana');
const usuario = localStorage.getItem('nombreUsuario');
console.log(usuario); // "Ana"

// Ejemplo de uso de Geolocation
navigator.geolocation.getCurrentPosition(pos => {
    console.log(`Lat: ${pos.coords.latitude}`);
    console.log(`Lon: ${pos.coords.longitude}`);
});

¡A Practicar! Abre la consola de desarrollador (F12), ve a "Consola" y pega el código de localStorage. Luego verifica en "Aplicación" que el dato persiste.

Accesibilidad y ARIA

Una página HTML completa debe ser usable por el mayor número posible de personas. Las etiquetas semánticas, junto con atributos ARIA, ayudan a los lectores de pantalla.

Roles y landmarks

Define regiones como banner, navigation o main

Navegación por teclado

Gestiona el foco con tabindex y aria-live

Texto alternativo

Describe imágenes e iconos con alt o aria-label

<header role="banner">
    <h1>Portal DAM</h1>
</header>

<nav aria-label="Navegación principal">
    <button aria-expanded="false" 
            aria-controls="menu" 
            aria-label="Abrir menú">
        <i class="fa-solid fa-bars" aria-hidden="true"></i>
    </button>
    <ul id="menu" hidden>
        <li><a href="#html">HTML</a></li>
        <li><a href="#css">CSS</a></li>
    </ul>
</nav>

<main role="main">
    <section aria-labelledby="seccion-form">
        <h2 id="seccion-form">Formulario de alta</h2>
        <form>
            <label for="email">Correo institucional</label>
            <input id="email" type="email" 
                   required aria-required="true">
        </form>
    </section>
</main>

¡A Practicar! Revisa cualquier formulario que hayas creado y añade etiquetas label, atributos aria-describedby y comprueba con Tab que el foco recorre los campos en orden lógico.

Plantillas y Componentes Reutilizables

Con HTML puedes crear bloques reutilizables usando elementos como template o personalizando etiquetas con Web Components.

<template>

Define fragmentos ocultos listos para clonar desde JavaScript

Slots

Los Web Components permiten insertar contenido dinámico

Includes

Reutiliza cabeceras mediante frameworks o motores de plantillas

<template id="card-template">
    <article class="curso">
        <h3 class="curso__titulo"></h3>
        <p class="curso__descripcion"></p>
    </article>
</template>

<section id="cursos"></section>

<script>
const cursos = [
    { titulo: 'Interfaces Web', 
      descripcion: 'HTML5 y componentes web.' },
    { titulo: 'Programación', 
      descripcion: 'Java y patrones orientados a objetos.' }
];

const tpl = document.getElementById('card-template');
const contenedor = document.getElementById('cursos');

cursos.forEach(curso => {
    const fragmento = tpl.content.cloneNode(true);
    fragmento.querySelector('.curso__titulo')
        .textContent = curso.titulo;
    fragmento.querySelector('.curso__descripcion')
        .textContent = curso.descripcion;
    contenedor.appendChild(fragmento);
});
</script>

¡A Practicar! Crea un template para mostrar fichas de asignaturas del ciclo DAM. Añade al menos tres asignaturas y genera el listado desde JavaScript.

Proyecto Integrador

Consolida lo aprendido construyendo una landing para un ciclo formativo DAM que combine estructura semántica, formularios y multimedia.

Paso a paso:

  1. Estructura general: usa header, main, section, aside y footer
  2. Sección de módulos: crea una tabla o cards para listar asignaturas
  3. Testimonios: agrega blockquote con citas de alumnado
  4. Formulario de inscripción: solicita nombre, email y turno preferido
  5. Llamada a la acción: botón que enlace a la página del centro

¡Entrega opcional! Publica tu landing en GitHub Pages o Netlify. Comparte el enlace con tus compañeros para recibir feedback.

Validación y Mejores Prácticas

Antes de dar por finalizado un documento HTML conviene pasar por un checklist que garantice calidad y mantenimiento.

  • Valida el marcado en validator.w3.org
  • Activa Lighthouse en Chrome DevTools
  • Comprueba que todos los enlaces funcionan
  • Asegúrate de que los formularios incluyen estados de error
  • Incluye comentarios puntuales para secciones complejas

Integración Web Completa

HTML es el esqueleto. CSS define el aspecto, JavaScript inyecta interactividad y XML transporta los datos. Juntos construyen experiencias web completas.

Estructura HTML

Organiza el contenido con secciones semánticas

Estilos CSS

Aplica diseño responsivo y animaciones

Lógica JavaScript

Consume XML/JSON y actualiza la vista

Datos XML

Centraliza información en formato interoperable

<!-- index.html -->
<section id="modulos" class="grid-modulos">
    <article class="modulo" data-tema="interfaces">
        <h3 class="modulo__titulo">Interfaces</h3>
        <ul class="modulo__lista" data-lista></ul>
    </article>
</section>

<link rel="stylesheet" href="modulos.css">
<script type="module" src="modulos.js"></script>
// js/modulos.js
const parser = new DOMParser();
const lista = document.querySelector('[data-lista]');

async function cargarModulos() {
    const respuesta = await fetch('datos/modulos.xml');
    const texto = await respuesta.text();
    const xml = parser.parseFromString(texto, 'application/xml');
    const modulos = [...xml.querySelectorAll('modulo')];

    lista.innerHTML = modulos.map(modulo => `
        <li>
            <span>${modulo.getAttribute('nombre')}</span>
            <small>${modulo.querySelector('horas').textContent} h</small>
        </li>
    `).join('');
}

cargarModulos();

    </modulo>
    <modulo nombre="Interfaces" ciclo="DAM2">
        <horas>140</horas>

    </modulo>
</plan>
