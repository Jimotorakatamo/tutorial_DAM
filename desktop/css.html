Introducción a CSS

El lenguaje CSS permite cambiar el aspecto de las páginas web. CSS son las siglas de Cascading Style Sheets (Hojas de estilo en cascada).

El W3C mantiene su estandarización (primera versión en 1996, actual CSS3).

Funcionamiento de CSS

CSS define reglas de presentación aplicadas a elementos HTML. Una regla se compone de:

  • Selector: indica a qué elementos se aplica.
  • Declaraciones: pares propiedad-valor (por ejemplo: color, fuente, espaciado).

Ejemplo de regla CSS

body {
  text-align: center;
  font-family: 'Roboto', sans-serif;
  background-color: black;
  color: white;
}
/* Este texto es un comentario */

Conectando HTML con CSS

Tres formas de aplicar CSS:

  • Interno: dentro de <style> en el <head>.
  • Externo: enlazando un archivo .css mediante <link>.
  • En línea: usando el atributo style= en una etiqueta.

Ejemplo inline:

<h1 style="color:blue; font-size:20px;">Hola!</h1>

Orden de prioridad:

  1. CSS inline → más fuerte
  2. CSS en <style>
  3. CSS externo → recomendado

Reglas CSS: conceptos básicos

  • Selectores: indican qué elementos se afectan.
  • Propiedades: definen qué aspecto cambiar.
  • Valores: definen el nuevo valor de esa propiedad.

Tipos de selectores

/* Elemento */
p { color: red; }

/* Universal */
* { margin: 0; }

/* Clase */
.nombreClase { color: blue; }

/* ID */
#miParrafo { text-align: center; }

/* Descendiente */
h1 em { color: red; }

/* Hijo */
div > p { ... }

/* Adyacente */
div + p { ... }

Unidades de medida

Absolutas

cm, mm, px, in (no varían)

Relativas

em, rem, %, vw, vh (se adaptan al dispositivo)

W3C recomienda em para tamaño de texto.

Colores y fondo

Formatos: nombre (red), hexadecimal (#FF0000), rgb(), rgba(), hsl(), hsla().

Propiedades: background-color, background-image, background-position, background-repeat.

body {
  background-image: url("fondo.png");
  background-position: 50% 50%;
}

Propiedades del texto

  • font-size: tamaño
  • font-style: normal | italic | oblique
  • font-weight: normal | bold | 100–900
  • font-family: tipo de letra

Alineación, interlineado, espaciado: controlados con propiedades de bloque.

Pseudo-clases y pseudo-elementos

Pseudo-clases

:hover, :first-child, :nth-child(), :focus, :visited

Pseudo-elementos

::before, ::after, ::first-line

Listas y tablas

Listas:

  • list-style-type: circle | square | none
  • list-style-position: inside | outside
  • list-style-image: url('icon.png')

Tablas:

  • caption-side: top | bottom
  • border-collapse: collapse
  • border-spacing: 5px
  • empty-cells: show | hide

Prioridad y cascada

  1. CSS inline
  2. <style> interno
  3. Archivo externo

Selectores más específicos tienen prioridad. En igualdad, gana el último leído.

¿Para qué sirve CSS?

CSS controla la apariencia y el comportamiento visual de tus interfaces. Define colores, tipografías, animaciones, layouts y cualquier detalle estético que transforma un esqueleto HTML en una experiencia pulida.

Interfaces web

Diseña paneles responsive, dashboards y tiendas con identidad propia.

Design systems

Construye bibliotecas de componentes reutilizables con tokens de estilo.

Aplicaciones híbridas

Personaliza apps PWA o vistas móviles usando la misma hoja de estilos.

Paso a paso

  1. Enlaza tu CSS: usa <link rel="stylesheet"> o @import si trabajas con compiladores.
  2. Define la base: ajusta tipografía, colores y un reset que normalice márgenes.
  3. Crea el layout: combina Flexbox y Grid para distribuir secciones y componentes.
  4. Refina interacciones: aplica transiciones, estados hover y variables para soportar temas.
:root {
    --color-fondo: #0f172a;
    --color-acento: #38bdf8;
}

.dashboard {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(260px, 1fr));
    gap: 1.5rem;
    padding: 2rem;
    background: radial-gradient(circle at 20% 20%, rgba(56, 189, 248, 0.18), transparent 55%),
                #0b1220;
}

.widget {
    background: rgba(15, 23, 42, 0.85);
    border: 1px solid rgba(56, 189, 248, 0.35);
    border-radius: 1rem;
    padding: 1.5rem;
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.widget:hover {
    transform: translateY(-6px);
    box-shadow: 0 22px 40px rgba(56, 189, 248, 0.28);
}

Flexbox

Flexbox es un modelo de diseño que permite crear layouts flexibles y responsivos de manera eficiente.

Dirección

Orientación del flujo

Justify

Alineación principal

Align

Alineación cruzada

.contenedor-flex {
    display: flex;
    justify-content: space-around; /* Distribuye los ítems */
    align-items: center; /* Centra verticalmente */
    height: 200px;
}

¡A Practicar!

Crea un <div> con la clase .galeria y pon tres imágenes dentro. Aplica display: flex; a la galería y observa cómo se alinean las imágenes. Prueba a cambiar justify-content a "center".

CSS Grid

Grid es un sistema de diseño bidimensional que permite crear layouts complejos con facilidad.

Columnas

Animaciones y Transiciones

Las animaciones y transiciones añaden dinamismo y mejoran la experiencia del usuario.

.boton-animado {

¡A Practicar!

Crea un botón en tu HTML. Añádele una transición para la propiedad transform. En el estado :hover, aplica transform: scale(1.1); para que crezca ligeramente al pasar el ratón por encima.

Variables CSS (Custom Properties)

Las variables CSS permiten almacenar valores reutilizables en tus hojas de estilo, facilitando la creación de sistemas de diseño y temas.

:root {
    --color-principal: #3498db;
    --fuente-base: 16px;
}

a {
    color: var(--color-principal);
}

¡A Practicar!

Define una variable CSS llamada --color-borde en :root y asígnale un color. Luego, úsala en la propiedad border-color de alguna de tus clases.

Pseudoclases y Pseudoelementos

Las pseudoclases y pseudoelementos son útiles para aplicar estilos en estados específicos de un elemento o para partes específicas de un elemento.

/* Estilo para el primer hijo de un elemento */
li:first-child {
    color: green;
}

/* Añade contenido antes de un elemento */
p::before {
    content: "► ";
}

¡A Practicar!

Usa la pseudoclase :nth-child(odd) en las filas de una tabla (tr) para darles un color de fondo ligeramente diferente y crear un efecto de "cebra".

Sombras y Gradientes

Las sombras y gradientes añaden profundidad y estilo a tus elementos.

.sombra {
    box-shadow: 5px 5px 15px rgba(0,0,0,0.2);
}

.gradiente {
    background: linear-gradient(to right, lightblue, navy);
}

¡A Practicar!

Aplica una sombra de texto (text-shadow) a tu h1 para darle un poco de profundidad. Por ejemplo: 2px 2px 4px #888888;.

Unidades en CSS

CSS ofrece una variedad de unidades para definir tamaños y distancias. Se dividen en absolutas y relativas.

Absolutas

px, pt, cm. Tienen un tamaño fijo y no cambian.

Relativas

%, em, rem, vw, vh. Su tamaño depende de otro elemento o del viewport.

Unidades Relativas Comunes

  • em: Relativa al tamaño de fuente del elemento padre.
  • rem (root em): Relativa al tamaño de fuente del elemento raíz (<html>). Es la más recomendada para crear layouts escalables.
  • vw (viewport width): 1vw es el 1% del ancho del viewport.
  • vh (viewport height): 1vh es el 1% del alto del viewport.
.absoluto { width: 200px; /* Píxeles */ }
.relativo { font-size: 1.2em; /* Relativo al padre */ }
.viewport { width: 50vw; /* 50% del ancho del viewport */ }
.flexible { flex-basis: 0; flex-grow: 1; /* Unidades Flexbox */ }

¡A Practicar!

Establece el tamaño de fuente del body en 16px. Luego, define el tamaño de fuente de un párrafo en 1.5rem. Observa cómo se calcula en base a la raíz (html), que por defecto es 16px.

Diseño Responsive Moderno

El objetivo del responsive es que el diseño se adapte fluidamente a cualquier pantalla. CSS moderno ofrece funciones como clamp(), consultas a nivel de contenedor y unidades fluidas.

Consultas de media (@media)

Define reglas específicas para breakpoints clave como móviles, tablets y escritorio.

Consultas de contenedor

Evalúan el tamaño del contenedor, ideales para componentes reutilizables.

Tipografía fluida

Usa clamp() y unidades vw para escalar títulos sin media queries extra.

:root {
    --max-width: clamp(320px, 90vw, 1100px);
}

.layout {
    width: min(var(--max-width), 100%);
    margin: 0 auto;
    padding: clamp(1.5rem, 3vw, 3rem);
}

h1 {
    font-size: clamp(2rem, 5vw, 3.6rem);
}

@media (max-width: 768px) {
    .sidebar {
        position: static;
        inset: auto;
    }
}

@container (min-width: 520px) {
    .tarjeta-curso {
        grid-template-columns: 1fr auto;
    }
}

¡A Practicar!

Convierte una sección rígida de tu proyecto en un layout fluido: establece un ancho máximo con min() o clamp(), añade padding proporcional y define al menos dos breakpoints (768px y 1024px).

Arquitectura y Mantenimiento

En proyectos de DAM o empresariales, organizar el CSS es clave para evitar conflictos. Patrones como BEM, ITCSS o la división por capas ayudan a mantener el código escalable.

BEM

Bloque–Elemento–Modificador para nombres de clases claros (.bloque__elemento--estado).

Capas

Define capas (@layer) para reset, componentes y utilidades.

Design tokens

Variables CSS centralizan colores, espaciados y tipografías.

@layer reset, base, components, utilities;

@layer reset {
    *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }
}

@layer base {
    :root {
        --color-brand: #2563eb;
        --espaciado-m: 1.5rem;
    }
    body { font-family: 'Poppins', sans-serif; line-height: 1.65; }
}

@layer components {
    .hero {
        padding: var(--espaciado-m);
        background: linear-gradient(135deg, var(--color-brand), #0f172a);
    }
    .hero__cta--destacado {
        background: #fff;
        color: var(--color-brand);
    }
}

@layer utilities {
    .text-center { text-align: center; }
    .mt-l { margin-top: 2.5rem; }
}

¡A Practicar!

Renombra un componente usando BEM (por ejemplo, .card → .card, .card__titulo, .card__cta--primaria) y agrupa sus estilos dentro de una capa @layer components.

Preprocesadores y Postprocesado

Sass, Less o PostCSS permiten escribir CSS más expresivo. Aunque luego se compile, conviene entender su sintaxis y beneficios.

Sass

Variables, mixins, anidación controlada y funciones personalizadas.

Autoprefixer

PostCSS añade prefijos automáticamente según el soporte del navegador.

Build tooling

Herramientas como Vite o Parcel empaquetan el CSS junto a JavaScript.

// Sass (SCSS)
$color-primario: #ec4899;

@mixin boton-base($bg, $color) {
    background: $bg;
    color: $color;
    padding: .75rem 1.5rem;
    border-radius: 999px;
}

.boton {
    @include boton-base($color-primario, #fff);
    &:hover { filter: brightness(1.05); }
}

/* CSS compilado */
.boton {
    background: #ec4899;
    color: #fff;
    padding: 0.75rem 1.5rem;
    border-radius: 999px;
}
.boton:hover {
    filter: brightness(1.05);
}

¡A Practicar!

Instala Sass de forma global (npm install -g sass) o usa la versión local. Crea un archivo estilos.scss con un mixin de botones y compílalo a CSS con sass estilos.scss estilos.css --watch.

CSS y JavaScript trabajando juntos

Las interfaces modernas combinan CSS con JavaScript para gestionar estados, temas y animaciones complejas.

Tema claro/oscuro

Alterna clases o atributos en <html> para cambiar variables CSS.

Estados interactivos

Usa classList.toggle() para activar modales o menús.

Preferencias del usuario

Lee matchMedia y sincroniza con localStorage.

<button id="toggle-layout">Cambiar layout</button>
<section class="panel panel--grid" id="panelCursos">...</section>

<script>
const toggle = document.getElementById('toggle-layout');
const panel = document.getElementById('panelCursos');

toggle.addEventListener('click', () => {
    panel.classList.toggle('panel--grid');
    panel.classList.toggle('panel--lista');
});
</script>

<style>
.panel--grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); gap: 1.5rem; }
.panel--lista { display: flex; flex-direction: column; gap: 1rem; }
</style>

¡A Practicar!

Añade un botón que permita activar/desactivar un modo de alto contraste: crea una clase .modo-contraste con fondos claros y texto oscuro, y alterna esa clase en body usando JavaScript.

Proyecto Integrador de Estilos

Aplica todo lo aprendido diseñando una intranet para el ciclo DAM con enfoque responsive, tokens de diseño y animaciones sutiles.

  1. Configura el tema: Define variables para colores primarios, secundarios y estados, además de espaciados y tipografía.
  2. Layout flexible: Usa Grid para el dashboard y Flexbox para el menú lateral.
  3. Componentes reutilizables: Diseña tarjetas de módulos, listados de tareas y un formulario modal.
  4. Feedback visual: Añade transiciones a los botones y estados :focus-visible accesibles.
  5. Dark mode: Implementa un toggle que cambie variables de color y sombras.

Herramientas de Diagnóstico

Optimiza tu CSS revisando peso, especificidad y rendimiento.

  • Abre Chrome DevTools > Coverage para detectar CSS no utilizado.
  • Utiliza la pestaña Accessibility para comprobar contraste y orden de tabulación.
  • Ejecuta Lighthouse e interpreta las recomendaciones sobre Best Practices y Performance.
  • Analiza el peso del CSS en la pestaña Network y considera minificar o dividir en chunks.
  • Pasa tu hoja de estilos por Stylelint para mantener un estilo consistente.

CSS conectado con HTML, JavaScript y XML

CSS da identidad visual al contenido HTML, responde a la lógica de JavaScript y presenta datos provenientes de XML. Mantener esta colaboración ordenada asegura escalabilidad y accesibilidad.

Variables temáticas

Sincroniza colores entre hojas globales y componentes.

Layouts dinámicos

Grid y Flex adaptan los datos XML que JavaScript inserta.

Estados reactivos

Clases toggled via JS disparan transiciones y microinteracciones.

js/modulos.js 
import { cargarPlan } from './servicios/xml.js';

const contenedor = document.querySelector('[data-modulos]');

cargarPlan('datos/modulos.xml').then(plan => {
    contenedor.innerHTML = plan.modulos.map(modulo => `
        <article class="modulo-card" data-estado="${modulo.destacado ? 'destacado' : 'normal'}">
            <h3>${modulo.nombre}</h3>
            <ul class="modulo-card__lista">
                <li><span>Horas</span><small>${modulo.horas}</small></li>
                <li><span>Profesorado</span><small>${modulo.profesor}</small></li>
            </ul>
        </article>
    `).join('');
});
servicios/xml.js 
export async function cargarPlan(ruta) {
    const respuesta = await fetch(ruta);
    const texto = await respuesta.text();
    const xml = new DOMParser().parseFromString(texto, 'application/xml');

    return {
        curso: xml.documentElement.getAttribute('curso'),
        modulos: [...xml.querySelectorAll('modulo')].map(modulo => ({
            nombre: modulo.getAttribute('nombre'),
            horas: modulo.querySelector('horas').textContent,
            profesor: modulo.querySelector('profesor').textContent,
            destacado: modulo.getAttribute('destacado') === 'true'
        }))
    };
}

¡A Practicar!

Define un set de variables y utilízalas para colorear tarjetas según los módulos que recibes por XML. Añade un toggle en JavaScript para destacar el módulo con más horas aplicando la clase data-estado="destacado".
