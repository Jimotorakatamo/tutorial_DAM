Aprende Python Moderno

Python es un lenguaje de programación versátil y fácil de aprender. En este tutorial, descubrirás los conceptos básicos, sintaxis, estructuras de control, funciones, manejo de archivos y mucho más.

Primeros pasos con Python

Python es único por su simplicidad y poder. Veamos algunos ejemplos básicos que demuestran su elegancia:

# El clásico "Hola, mundo"
print("¡Hola, mundo!")

# Interacción con el usuario
nombre = input("¿Cómo te llamas? ")
print(f"¡Bienvenido/a a Python, {nombre}!")

# Una pequeña calculadora
numero1 = float(input("Introduce el primer número: "))
numero2 = float(input("Introduce el segundo número: "))

print(f"""
Operaciones básicas con {numero1} y {numero2}:
Suma: {numero1 + numero2}
Resta: {numero1 - numero2}
Multiplicación: {numero1 * numero2}
División: {numero1 / numero2}
""")

Simple y Legible

Sintaxis clara que parece pseudocódigo

Productivo

Haz más con menos código

Versátil

Web, IA, datos, juegos y más

¿Cómo programar en Python?

Comenzar con Python es un proceso sencillo. Te guiaremos paso a paso para configurar tu entorno de desarrollo perfecto:

1. Instala Python

  • Descarga Python desde python.org
  • Marca la opción "Add Python to PATH" durante la instalación
  • Verifica la instalación en la terminal:
python --version
pip --version

2. Elige tu IDE

Recomendamos estas excelentes opciones:

  • Visual Studio Code + Python Extension
    • Ligero y personalizable
    • Gran soporte para debugging
    • Integración con Git
  • PyCharm
    • IDE completo para Python
    • Potentes herramientas de refactorización
    • Ideal para proyectos grandes

3. Tu primer proyecto

  1. Crea una carpeta para tu proyecto
  2. Abre tu editor favorito
  3. Crea un archivo mi_programa.py
  4. Escribe tu código:
# mi_programa.py
def calcular_edad_en_dias(edad):
    return edad * 365

mi_edad = int(input("¿Cuál es tu edad? "))
dias = calcular_edad_en_dias(mi_edad)
print(f"¡Has vivido aproximadamente {dias:,} días!")

4. Mejores prácticas

  • Usa entornos virtuales para cada proyecto: 
    python -m venv mi_entorno
mi_entorno\Scripts\activate  # Windows
pip install nombre_paquete
  • Sigue la guía de estilo PEP 8
  • Escribe pruebas unitarias desde el principio
  • Usa control de versiones (Git)

Sintaxis básica

Python destaca por su sintaxis limpia y legible. No necesitas punto y coma ni llaves - la indentación define los bloques de código.

Indentación

Define bloques de código con espacios

Comentarios

Usa # para comentar código

Líneas

Una instrucción por línea

# Ejemplos de sintaxis básica
def calcular_descuento(precio, porcentaje):
    """Esta función calcula el descuento de un precio"""
    descuento = precio * (porcentaje / 100)
    precio_final = precio - descuento
    return precio_final

# Uso de la función con valores
precio_original = 100
descuento = 20
precio_con_descuento = calcular_descuento(precio_original, descuento)
print(f"Precio final con {descuento}% de descuento: ${precio_con_descuento}")

Python utiliza tipado dinámico, lo que significa que las variables pueden cambiar de tipo y no necesitas declararlas explícitamente.

Números

int, float, complex

Texto

str (cadenas)

Booleanos

True o False

Colecciones

list, tuple, dict, set

# Ejemplos de tipos de datos
# Números
edad = 25              # int (entero)

Estructuras de control

Las estructuras de control determinan el flujo de ejecución de tu programa. Python ofrece una sintaxis clara y concisa para cada tipo de control.

if/elif/else

Decisiones condicionales

while

Bucles con condición

for

Iteración sobre secuencias

# Ejemplos de estructuras de control

# if/elif/else
def calificar_nota(nota):
    if nota >= 90:
        return "Sobresaliente"
    elif nota >= 70:
        return "Notable"
    elif nota >= 60:
        return "Aprobado"
    else:
        return "Necesita mejorar"

# Bucle for con range y enumerate
print("Tabla de multiplicar del 5:")
for i in range(1, 11):
    print(f"5 x {i} = {5 * i}")

# Bucle while con break y continue
numero = 0
while True:
    numero += 1
    if numero == 3:
        continue  # Salta esta iteración
    if numero > 5:
        break    # Sale del bucle
    print(numero)

Funciones

Las funciones son bloques de código reutilizables que pueden tomar argumentos y devolver resultados.

Argumentos

Parámetros de entrada

Return

Valor de retorno

Args/Kwargs

Argumentos variables

# Ejemplos de funciones

# Función con argumentos por defecto
def saludar(nombre, saludo="¡Hola"):
    return f"{saludo}, {nombre}!"

# Función con argumentos variables
def sumar(*args, **kwargs):
    """
    args: tupla de argumentos posicionales
    kwargs: diccionario de argumentos nombrados
    """
    total = sum(args)  # suma los argumentos posicionales
    if 'extra' in kwargs:
        total += kwargs['extra']
    return total

# Ejemplos de uso
print(saludar("Ana"))  # ¡Hola, Ana!
print(saludar("Juan", "Buenos días"))  # Buenos días, Juan!

# Usando args y kwargs
print(sumar(1, 2, 3))  # 6
print(sumar(1, 2, extra=3))  # 6

Manejo de archivos

Python proporciona una forma sencilla y segura de trabajar con archivos usando el contexto 'with'.

Escritura

Modo 'w' para escribir

Lectura

Modo 'r' para leer

Append

Modo 'a' para añadir

# Ejemplos de manejo de archivos

# Escribir en un archivo
def guardar_notas(archivo, notas):
    with open(archivo, 'w', encoding='utf-8') as f:
        for estudiante, nota in notas.items():
            f.write(f"{estudiante}: {nota}\n")

# Leer de un archivo
def leer_notas(archivo):
    notas = {}
    with open(archivo, 'r', encoding='utf-8') as f:
        for linea in f:
            estudiante, nota = linea.strip().split(': ')
            notas[estudiante] = float(nota)
    return notas

# Ejemplo de uso
notas_estudiantes = {
    'Ana': 9.5,
    'Juan': 8.7,
    'María': 9.2
}

# Guardar notas
guardar_notas('notas.txt', notas_estudiantes)

# Leer notas
notas_leidas = leer_notas('notas.txt')
print("Notas leídas:", notas_leidas)

Las listas y diccionarios son estructuras de datos versátiles y fundamentales en Python.

Listas

Colecciones ordenadas

Diccionarios

Pares clave-valor

Operaciones útiles

# Ejemplos de listas y diccionarios

# Operaciones con listas
frutas = ['manzana', 'plátano', 'naranja']
frutas.append('pera')              # Añadir al final
frutas.insert(1, 'uva')           # Insertar en posición
frutas.remove('plátano')          # Eliminar elemento

# List comprehension
numeros = [1, 2, 3, 4, 5]
cuadrados = [n ** 2 for n in numeros]  # [1, 4, 9, 16, 25]

# Operaciones con diccionarios
persona = {
    'nombre': 'Ana',
    'edad': 25,
    'ciudad': 'Madrid'
}

# Acceder y modificar
print(persona.get('nombre', 'No encontrado'))
persona['profesion'] = 'Ingeniera'

# Dict comprehension
cuadrados_dict = {n: n**2 for n in range(5)}  # {0:0, 1:1, 2:4, 3:9, 4:16}

Clases y Objetos

Python es un lenguaje orientado a objetos. Aquí un ejemplo de cómo crear y usar clases:

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    
    def saludar(self):
        return f"¡Hola! Soy {self.nombre}"

# Crear y usar un objeto
ana = Persona("Ana", 25)
print(ana.saludar())

Manejo de Excepciones

Python proporciona un sistema robusto para manejar errores:

try:
    numero = int(input("Introduce un número: "))
    resultado = 10 / numero
except ValueError:
    print("Error: Debes introducir un número")
except ZeroDivisionError:
    print("Error: No se puede dividir por cero")
else:
    print(f"Resultado: {resultado}")
finally:
    print("Proceso completado")

Funciones Avanzadas y Programación Funcional

Python soporta paradigmas de programación funcional que hacen el código más declarativo y conciso.

Funciones Lambda

Pequeñas funciones anónimas definidas con la palabra clave lambda.

Map, Filter, Reduce

Funciones de orden superior para operar sobre listas y otros iterables.

Decoradores

Funciones que modifican o envuelven a otras funciones para extender su comportamiento.

# Función Lambda para sumar
suma = lambda x, y: x + y
print(suma(3, 5))  # Salida: 8

# Uso de map y filter
numeros = [1, 2, 3, 4, 5]
cuadrados = list(map(lambda x: x**2, numeros))  # [1, 4, 9, 16, 25]
pares = list(filter(lambda x: x % 2 == 0, numeros))  # [2, 4]

# Decorador simple para medir tiempo de ejecución
import time

def cronometro(func):
    def wrapper(*args, **kwargs):
        inicio = time.time()
        resultado = func(*args, **kwargs)
        fin = time.time()
        print(f"'{func.__name__}' tardó {fin - inicio:.4f} segundos.")
        return resultado
    return wrapper

@cronometro
def proceso_largo():
    time.sleep(1)
    print("Proceso finalizado.")

proceso_largo()

Programación Orientada a Objetos (POO)

La POO en Python permite modelar problemas complejos mediante clases y objetos, fomentando la reutilización y organización del código.

class Vehiculo:
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo
        self.arrancado = False

    def arrancar(self):
        self.arrancado = True
        print(f"El {self.marca} {self.modelo} ha arrancado.")

    def describir(self):
        return f"Vehículo: {self.marca} {self.modelo}"

class Coche(Vehiculo):  # Herencia de Vehiculo
    def __init__(self, marca, modelo, num_puertas):
        super().__init__(marca, modelo)
        self.num_puertas = num_puertas

    def describir(self):  # Polimorfismo (sobrescritura de método)
        return f"Coche de {self.num_puertas} puertas: {self.marca} {self.modelo}"

# Creación de objetos
mi_coche = Coche("Tesla", "Model 3", 4)
mi_coche.arrancar()
print(mi_coche.describir())

Módulos y Paquetes

Python utiliza módulos (archivos .py) y paquetes (directorios con un __init__.py) para organizar el código en proyectos grandes.

# Importar un módulo de la librería estándar
import math
print(f"La raíz cuadrada de 16 es {math.sqrt(16)}")

# Importar un objeto específico de un módulo
from datetime import datetime
print(f"Fecha y hora actual: {datetime.now()}")

# Importar un paquete de terceros (previamente instalado con 'pip install requests')
import requests
response = requests.get("https://api.github.com")
print(f"Respuesta de GitHub: {response.status_code}")
