Introducción a JavaScript

JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web. Cada vez que una página web hace más que simplemente mostrar información estática, JavaScript probablemente está involucrado.

// Muestra una alerta en el navegador
alert('¡Hola, Mundo!');

// Escribe en la consola del navegador
console.log('Este es un mensaje para desarrolladores.');

¡A Practicar!

Abre la consola de tu navegador (F12) y escribe alert('Mi primer script');. Luego, intenta mostrar tu nombre en la consola usando console.log().

Variables y Tipos de Datos

JavaScript es un lenguaje dinámico con diferentes formas de declarar variables y múltiples tipos de datos.

let nombre = "Juan"; // String
const edad = 30; // Number
let esEstudiante = true; // Boolean
let hobbies = ["leer", "correr"]; // Array
let persona = { nombre: "Ana", edad: 25 }; // Object

¡A Practicar!

Declara una variable `let` para tu comida favorita y una constante `const` para tu año de nacimiento. Muestra ambas en la consola.

Operadores

Los operadores son símbolos que realizan operaciones sobre variables y valores.

let suma = 10 + 5; // 15
let esMayor = suma > 10; // true
let yLogico = esMayor && (suma < 20); // true
let oLogico = (suma === 15) || (nombre === "Pedro"); // true

¡A Practicar!

Crea dos variables numéricas. Calcula su multiplicación y guárdala en una tercera variable. Luego, comprueba si el resultado es un número par usando el operador módulo %.

Condicionales y Bucles

Las estructuras de control te permiten manejar la lógica de tu programa: tomar decisiones y repetir acciones.

if (edad >= 18) {
    console.log("Es mayor de edad.");
} else {
    console.log("Es menor de edad.");
}

for (let i = 0; i < 3; i++) {
    console.log("Iteración número " + i);
}

¡A Practicar!

Escribe un bucle for que imprima los números del 1 al 5. Dentro del bucle, usa un if para imprimir "¡Cinco!" solo cuando el número sea 5.

Funciones

Las funciones son bloques de código reutilizables que pueden recibir parámetros y devolver valores.

function saludar(nombre) {
    return "Hola, " + nombre;
}

const saludo = saludar("Mundo");
console.log(saludo); // "Hola, Mundo"

¡A Practicar!

Crea una función llamada calcularArea que acepte dos parámetros: ancho y alto. La función debe devolver el área (ancho * alto). Llama a la función con dos números y muestra el resultado en la consola.

Manipulación del DOM

El DOM (Document Object Model) es la representación en memoria de la estructura HTML de tu página.

// Seleccionar un elemento por su ID
const titulo = document.getElementById('titulo-principal');

// Cambiar su contenido de texto
titulo.textContent = 'Nuevo Título desde JS';

// Añadir un evento de clic
titulo.addEventListener('click', () => {
    alert('¡Hiciste clic en el título!');
});

¡A Practicar!

Añade un botón a tu HTML con un ID único. Desde JavaScript, selecciónalo y añádele un `addEventListener` para que, al hacer clic, cambie el color de fondo de la página (`document.body.style.backgroundColor`).

Eventos

Los eventos son acciones que ocurren en la página web y que puedes detectar y manejar con JavaScript.

const miBoton = document.querySelector('#miBoton');

miBoton.addEventListener('mouseover', () => {
    console.log('El ratón está sobre el botón.');
});

window.addEventListener('keydown', (evento) => {
    console.log(`Tecla presionada: ${evento.key}`);
});

¡A Practicar!

Añade un campo de texto (input) a tu HTML. Usa `addEventListener` para escuchar el evento `input` y muestra en la consola el valor actual del campo (`evento.target.value`) mientras el usuario escribe.

Programación Asíncrona

JavaScript maneja operaciones asíncronas mediante Promesas, async/await y callbacks.

fetch('https://api.github.com/users/github')
    .then(respuesta => respuesta.json())
    .then(datos => {
        console.log(datos.name); // Muestra el nombre del usuario
    })
    .catch(error => console.error('Error:', error));

¡A Practicar!

Usa la API `fetch` para obtener datos de la URL pública: `https://jsonplaceholder.typicode.com/todos/1`. Muestra el título (`title`) de la tarea obtenida en la consola.

Nuevas Características de ES6+

ES6+ trajo muchas mejoras a JavaScript, incluyendo nuevas sintaxis y funcionalidades.

// Funciones de flecha
const sumar = (a, b) => a + b;

// Template literals
const mensaje = `La suma es ${sumar(5, 3)}`;

// Desestructuración
const { nombre, edad } = persona;

¡A Practicar!

Crea un objeto con dos propiedades: `ciudad` y `pais`. Luego, usa la desestructuración para extraer esos valores en dos variables separadas y muéstralas en la consola usando un *template literal*.

Módulos ES6

Los módulos permiten organizar y reutilizar código dividiéndolo en archivos separados.

// En archivo: utilidades.js
export const PI = 3.1416;

// En archivo: main.js
import { PI } from './utilidades.js';
console.log(PI);

¡A Practicar!

Crea dos archivos JS. En uno, exporta una función que salude. En el otro, impórtala y úsala. Recuerda que para probar esto en el navegador, tus etiquetas <script> en HTML deben tener type="module".

JSON: JavaScript Object Notation

JSON es un formato de texto ligero para el intercambio de datos. Es muy fácil de usar en JavaScript.

// Convertir objeto a string JSON
const jsonString = JSON.stringify(persona);

// Convertir string JSON a objeto
const objetoDesdeJson = JSON.parse(jsonString);

¡A Practicar!

Crea un objeto JavaScript que represente un libro (con título y autor). Conviértelo a una cadena JSON y muéstrala en la consola. Luego, vuelve a convertir esa cadena en un objeto y muestra el título del libro.
