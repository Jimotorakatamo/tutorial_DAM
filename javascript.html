Introducción a JavaScript

JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web. Cada vez que una página web hace más que simplemente mostrar información estática, JavaScript probablemente está involucrado.

// Muestra una alerta en el navegador
alert('¡Hola, Mundo!');

// Escribe en la consola del navegador
console.log('Este es un mensaje para desarrolladores.');

¡A Practicar!

Abre la consola de tu navegador (F12) y escribe alert('Mi primer script');. Luego, intenta mostrar tu nombre en la consola usando console.log().

¿Para qué sirve JavaScript?

JavaScript aporta dinamismo, lógica y conectividad a la web. Corre en el navegador para reaccionar a eventos del usuario y en el servidor (Node.js) para construir APIs y automatizaciones.

Interfaces reactivas

Dashboards, SPA y formularios que responden al instante sin recargar la página.

Servicios backend

APIs REST, WebSockets y microservicios escritos con Node.js o Deno.

Automatización

Scripts para build, pruebas end-to-end y tareas de integración continua.

Paso a paso

  1. Identifica la interacción: decide qué evento o dato debe disparar tu lógica.
  2. Obtén referencias del DOM: usa querySelector para manipular elementos.
  3. Procesa datos: transforma información con funciones puras o llamadas a APIs.
  4. Actualiza la vista: modifica texto, clases o renderiza plantillas de forma segura.
const listaCursos = document.querySelector('[data-cursos]');
const filtro = document.querySelector('#filtro-modalidad');

async function cargarCursos() {
    const respuesta = await fetch('/api/cursos');
    const cursos = await respuesta.json();
    renderizar(cursos);
}

function renderizar(datos) {
    listaCursos.innerHTML = datos.map((curso) => `
        <article class="curso" data-modalidad="${curso.modalidad}">
            <h3>${curso.nombre}</h3>
            <p>${curso.horas} horas · ${curso.modalidad}</p>
        </article>
    `).join('');
}

filtro.addEventListener('change', (evento) => {
    const modalidad = evento.target.value;
    const cursosFiltrados = [...listaCursos.querySelectorAll('.curso')]
        .filter((card) => modalidad === 'todas' || card.dataset.modalidad === modalidad);

    listaCursos.innerHTML = cursosFiltrados.map((nodo) => nodo.outerHTML).join('');
});

cargarCursos();

Variables y Tipos de Datos

JavaScript es un lenguaje dinámico con diferentes formas de declarar variables y múltiples tipos de datos.

let nombre = "Juan"; // String
const edad = 30; // Number
let esEstudiante = true; // Boolean
let hobbies = ["leer", "correr"]; // Array
let persona = { nombre: "Ana", edad: 25 }; // Object

¡A Practicar!

Declara una variable `let` para tu comida favorita y una constante `const` para tu año de nacimiento. Muestra ambas en la consola.

Operadores

Los operadores son símbolos que realizan operaciones sobre variables y valores.

let suma = 10 + 5; // 15
let esMayor = suma > 10; // true
let yLogico = esMayor && (suma < 20); // true
let oLogico = (suma === 15) || (nombre === "Pedro"); // true

¡A Practicar!

Crea dos variables numéricas. Calcula su multiplicación y guárdala en una tercera variable. Luego, comprueba si el resultado es un número par usando el operador módulo %.

Condicionales y Bucles

Las estructuras de control te permiten manejar la lógica de tu programa: tomar decisiones y repetir acciones.

if (edad >= 18) {
    console.log("Es mayor de edad.");
} else {
    console.log("Es menor de edad.");
}

for (let i = 0; i < 3; i++) {
    console.log("Iteración número " + i);
}

¡A Practicar!

Escribe un bucle for que imprima los números del 1 al 5. Dentro del bucle, usa un if para imprimir "¡Cinco!" solo cuando el número sea 5.

Funciones

Las funciones son bloques de código reutilizables que pueden recibir parámetros y devolver valores.

function saludar(nombre) {
    return "Hola, " + nombre;
}

const saludo = saludar("Mundo");
console.log(saludo); // "Hola, Mundo"

¡A Practicar!

Crea una función llamada calcularArea que acepte dos parámetros: ancho y alto. La función debe devolver el área (ancho * alto). Llama a la función con dos números y muestra el resultado en la consola.

Manipulación del DOM

El DOM (Document Object Model) es la representación en memoria de la estructura HTML de tu página.

// Seleccionar un elemento por su ID
const titulo = document.getElementById('titulo-principal');

// Cambiar su contenido de texto
titulo.textContent = 'Nuevo Título desde JS';

// Añadir un evento de clic
titulo.addEventListener('click', () => {
    alert('¡Hiciste clic en el título!');
});

¡A Practicar!

Añade un botón a tu HTML con un ID único. Desde JavaScript, selecciónalo y añádele un `addEventListener` para que, al hacer clic, cambie el color de fondo de la página (`document.body.style.backgroundColor`).

Eventos

Los eventos son acciones que ocurren en la página web y que puedes detectar y manejar con JavaScript.

const miBoton = document.querySelector('#miBoton');

miBoton.addEventListener('mouseover', () => {
    console.log('El ratón está sobre el botón.');
});

window.addEventListener('keydown', (evento) => {
    console.log(`Tecla presionada: ${evento.key}`);
});

¡A Practicar!

Añade un campo de texto (input) a tu HTML. Usa `addEventListener` para escuchar el evento `input` y muestra en la consola el valor actual del campo (`evento.target.value`) mientras el usuario escribe.

Programación Asíncrona

JavaScript maneja operaciones asíncronas mediante Promesas, async/await y callbacks.

fetch('https://api.github.com/users/github')
    .then(respuesta => respuesta.json())
    .then(datos => {
        console.log(datos.name); // Muestra el nombre del usuario
    })
    .catch(error => console.error('Error:', error));

¡A Practicar!

Usa la API `fetch` para obtener datos de la URL pública: `https://jsonplaceholder.typicode.com/todos/1`. Muestra el título (`title`) de la tarea obtenida en la consola.

Nuevas Características de ES6+

ES6+ trajo muchas mejoras a JavaScript, incluyendo nuevas sintaxis y funcionalidades.

// Funciones de flecha
const sumar = (a, b) => a + b;

// Template literals
const mensaje = `La suma es ${sumar(5, 3)}`;

// Desestructuración
const { nombre, edad } = persona;

¡A Practicar!

Crea un objeto con dos propiedades: `ciudad` y `pais`. Luego, usa la desestructuración para extraer esos valores en dos variables separadas y muéstralas en la consola usando un *template literal*.

Módulos ES6

Los módulos permiten organizar y reutilizar código dividiéndolo en archivos separados.

// En archivo: utilidades.js
export const PI = 3.1416;

// En archivo: main.js
import { PI } from './utilidades.js';
console.log(PI);

¡A Practicar!

Crea dos archivos JS. En uno, exporta una función que salude. En el otro, impórtala y úsala. Recuerda que para probar esto en el navegador, tus etiquetas <script> en HTML deben tener type="module".

JSON: JavaScript Object Notation

JSON es un formato de texto ligero para el intercambio de datos. Es muy fácil de usar en JavaScript.

// Convertir objeto a string JSON
const jsonString = JSON.stringify(persona);

// Convertir string JSON a objeto
const objetoDesdeJson = JSON.parse(jsonString);

Estructuras de Datos Modernas

Desde ES6 disponemos de colecciones enriquecidas como Map, Set o iteradores. Entenderlas permite escribir código más expresivo y eficiente.

Map

Pares clave/valor con cualquier tipo como clave.

Set

Almacena valores únicos, ideal para eliminar duplicados.

Iterables

Permiten usar for...of, el operador spread y destructuración.

const roles = new Map();
roles.set('gonzalo', 'admin');
roles.set('maria', 'editor');

for (const [usuario, rol] of roles) {
    console.log(usuario, rol);
}

const tecnologias = ['HTML', 'CSS', 'JS', 'CSS'];
const tecnologiasUnicas = new Set(tecnologias);
console.log([...tecnologiasUnicas]); // ['HTML', 'CSS', 'JS']

const curso = {
    nombre: 'DAM',
    modulos: ['Programación', 'Interfaces', 'Bases de datos']
};

for (const modulo of curso.modulos) {
    console.log(modulo);
}

¡A Practicar!

Crea un Set con los lenguajes que has estudiado y conviértelo a array con el operador spread. Luego, genera un Map que relacione cada lenguaje con el nivel que sientes (básico, medio, avanzado).

Programación Orientada a Objetos

JavaScript soporta clases desde ES6, pero su modelo combina prototipos y azúcar sintáctico. Dominarlo ayuda a estructurar aplicaciones grandes.

class Curso {
    constructor(nombre, horas) {
        this.nombre = nombre;
        this.horas = horas;
    }

    resumen() {
        return `${this.nombre} (${this.horas} h)`;
    }
}

class CursoOnline extends Curso {
    #plataforma;

    constructor(nombre, horas, plataforma) {
        super(nombre, horas);
        this.#plataforma = plataforma;
    }

    resumen() {
        return `${super.resumen()} · ${this.#plataforma}`;
    }
}

const dam = new CursoOnline('Desarrollo de Interfaces', 96, 'Aula Virtual');
console.log(dam.resumen());

Integración con APIs REST

El ecosistema DAM incluye backends en Java o Python. Desde JavaScript consumimos APIs mediante fetch o librerías como Axios.

async function cargarCursos() {
    try {
        const respuesta = await fetch('https://dummyjson.com/products?limit=4');
        if (!respuesta.ok) throw new Error('Petición fallida');
        const datos = await respuesta.json();
        const lista = document.querySelector('#listaCursos');
        datos.products.forEach(producto => {
            const item = document.createElement('li');
            item.textContent = `${producto.title} · $${producto.price}`;
            lista.appendChild(item);
        });
    } catch (error) {
        console.error('Error al cargar cursos', error);
    }
}

cargarCursos();

Almacenamiento en el Navegador

Mantén preferencias y datos offline usando LocalStorage, SessionStorage o IndexedDB.

// Guardar configuración
localStorage.setItem('tema', 'oscuro');

// Recuperar configuración
const temaAlmacenado = localStorage.getItem('tema') ?? 'claro';

// IndexedDB básico
const solicitud = indexedDB.open('dam-db', 1);
solicitud.onupgradeneeded = (evento) => {
    const db = evento.target.result;
    db.createObjectStore('alumnos', { keyPath: 'id', autoIncrement: true });
};

solicitud.onsuccess = () => {
    const db = solicitud.result;
    const transaccion = db.transaction('alumnos', 'readwrite');
    transaccion.objectStore('alumnos').add({ nombre: 'Gonzalo', curso: 'DAM1' });
};

¡A Practicar!

Testing y Calidad

Asegura la calidad de tu código con pruebas unitarias y linters. Aunque sean conceptos de frameworks, puedes iniciar con herramientas sencillas.

  • Instala npm init -y y añade npm install --save-dev vitest. Crea un test simple para una función suma().
  • Ejecuta npx eslint --init y configura reglas para código consistente (usa el estilo recomendado).
  • Integra pruebas en tu flujo CI/CD básico usando GitHub Actions o scripts de npm.
  • Utiliza Chrome DevTools > Sources > Snippets para testear fragmentos rápidamente.

import { leerPlanFormativo } from './servicios/xml.js';

const seccion = document.querySelector('[data-modulos]');
const plantilla = document.getElementById('card-modulo');

leerPlanFormativo('datos/modulos.xml').then(plan => {
    const fragmento = document.createDocumentFragment();

    plan.modulos.forEach(modulo => {
        const nodo = plantilla.content.cloneNode(true);
        nodo.querySelector('.modulo-card__titulo').textContent = modulo.nombre;
        nodo.querySelector('.modulo-card__horas').textContent = `${modulo.horas} horas · ${modulo.profesor}`;
        nodo.querySelector('.modulo-card').dataset.estado = modulo.nuevo ? 'nuevo' : 'existente';
        fragmento.appendChild(nodo);
    });

    seccion.replaceChildren(fragmento);
});
servicios/xml.js 
export async function leerPlanFormativo(ruta) {
    const respuesta = await fetch(ruta);
    const texto = await respuesta.text();
    const xml = new DOMParser().parseFromString(texto, 'application/xml');

    return {
        curso: xml.documentElement.getAttribute('curso'),
        modulos: [...xml.querySelectorAll('modulo')].map(nodo => ({
            nombre: nodo.getAttribute('nombre'),
            horas: Number(nodo.querySelector('horas').textContent),
            profesor: nodo.querySelector('profesor').textContent,
            nuevo: nodo.getAttribute('nuevo') === 'true'
        }))
    };
}
