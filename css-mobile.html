Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje que define el aspecto visual de las páginas web.

Primera versión: 1996
Versión actual: CSS3
Mantenido por: W3C

¿Qué hace CSS?

CSS define reglas de presentación que se aplican a elementos HTML:

  • Selector: indica qué elementos afectar
  • Declaraciones: pares propiedad-valor

Ejemplo básico

body {
  text-align: center;
  font-family: 'Roboto', sans-serif;
  background-color: black;
  color: white;
}
/* Esto es un comentario */

Formas de aplicar CSS

Interno

Dentro de <style> en el <head>

Externo

Archivo .css enlazado con <link>

En línea

Atributo style="" en la etiqueta

Ejemplo inline

<h1 style="color:blue; font-size:1.25em;">
  Hola Mundo
</h1>
Orden de prioridad:
  1. CSS inline (más fuerte)
  2. CSS en <style>
  3. CSS externo (recomendado)

Selectores CSS

Los selectores indican qué elementos HTML serán afectados por las reglas CSS.

Tipos de selectores

/* Selector de elemento */
p { color: red; }

/* Selector universal */
* { margin: 0; padding: 0; }

/* Selector de clase */
.miClase { color: blue; }

/* Selector de ID */
#miID { text-align: center; }

/* Selector descendiente */
div p { color: green; }

/* Selector hijo directo */
div > p { font-weight: bold; }

/* Selector adyacente */
h1 + p { margin-top: 0; }

/* Selector de atributo */
input[type="text"] { ... }
Buena práctica: Usa clases para estilos reutilizables e IDs solo para elementos únicos.

Pseudoclases

Definen estados especiales de elementos:

/* Al pasar el ratón */
a:hover { color: red; }

/* Primer hijo */
li:first-child { font-weight: bold; }

/* Último hijo */
li:last-child { color: blue; }

/* Elemento n-ésimo */
tr:nth-child(even) { background: #f0f0f0; }

/* Al hacer focus */
input:focus { border-color: blue; }

Pseudoelementos

Estilizan partes específicas de un elemento:

/* Primera letra */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

/* Primera línea */
p::first-line { color: blue; }

/* Antes del contenido */
.icono::before {
  content: "→ ";
  color: red;
}

/* Después del contenido */
.icono::after {
  content: " ✓";
}

Modelo de Cajas

Cada elemento HTML es una caja rectangular compuesta por:

Content

Contenido del elemento (texto, imágenes, etc.)

Padding

Espacio entre contenido y borde (transparente)

Border

Borde alrededor del padding (puede tener color/grosor)

Margin

Espacio exterior entre elementos (transparente)

.caja {
  /* Contenido */
  width: 18.75em;
  height: 12.5em;
  
  /* Padding */
  padding: 1.25em;
  /* También: padding-top, padding-right, etc. */
  
  /* Border */
  border: 0.125em solid black;
  border-radius: 0.625em;
  
  /* Margin */
  margin: 1em;
  /* También: margin-top, margin-bottom, etc. */
}

Box-sizing

Controla cómo se calculan ancho y alto:

/* content-box (por defecto) */
/* width/height solo afectan al contenido */
.caja1 { box-sizing: content-box; }

/* border-box (recomendado) */
/* width/height incluyen padding y border */
.caja2 { box-sizing: border-box; }

/* Aplicar a todo el sitio */
* {
  box-sizing: border-box;
}
Tip profesional: Usa box-sizing: border-box en todos tus proyectos para facilitar el diseño responsive.

Display

Define cómo se comporta un elemento en el flujo:

/* Bloque: ocupa todo el ancho */
div { display: block; }

/* En línea: fluye con el texto */
span { display: inline; }

/* En línea-bloque: híbrido */
img { display: inline-block; }

/* Ocultar elemento */
.oculto { display: none; }

/* Flexbox */
.contenedor { display: flex; }

/* Grid */
.grid { display: grid; }

Texto y Fuentes

Propiedades de texto

p {
  /* Color */
  color: #333333;
  
  /* Alineación */
  text-align: center; /* left, right, justify */
  
  /* Decoración */
  text-decoration: underline;
  
  /* Transformación */
  text-transform: uppercase; /* lowercase, capitalize */
  
  /* Espaciado entre letras */
  letter-spacing: 0.125em;
  
  /* Espaciado entre palabras */
  word-spacing: 0.3125em;
  
  /* Altura de línea */
  line-height: 1.6;
  
  /* Sombra de texto */
  text-shadow: 0.125em 0.125em 0.25em rgba(0,0,0,0.3);
}

Fuentes

body {
  /* Familia de fuentes */
  font-family: 'Arial', 'Helvetica', sans-serif;
  
  /* Tamaño */
  font-size: 1em; /* También: em, rem, % */
  
  /* Peso (grosor) */
  font-weight: 700; /* 100-900 o bold, normal */
  
  /* Estilo */
  font-style: italic; /* normal, oblique */
  
  /* Variante */
  font-variant: small-caps;
}

Google Fonts

Importar fuentes personalizadas:

/* En el HTML */
<link rel="preconnect" 
      href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2
      ?family=Poppins:wght@300;400;600&display=swap" 
      rel="stylesheet">

/* En el CSS */
body {
  font-family: 'Poppins', sans-serif;
}
Unidades recomendadas:
  • rem: relativo al tamaño raíz
  • em: relativo al padre
  • Evita px para mejor accesibilidad

Colores y Fondos

Formatos de color

/* Nombre de color */
color: red;

/* Hexadecimal */
color: #ff0000;
color: #f00; /* forma corta */

/* RGB */
color: rgb(255, 0, 0);

/* RGBA (con transparencia) */
color: rgba(255, 0, 0, 0.5);

/* HSL (tono, saturación, luminosidad) */
color: hsl(0, 100%, 50%);

/* HSLA (con transparencia) */
color: hsla(0, 100%, 50%, 0.8);

Fondos

.elemento {
  /* Color sólido */
  background-color: #f0f0f0;
  
  /* Imagen */
  background-image: url('imagen.jpg');
  
  /* Repetición */
  background-repeat: no-repeat; /* repeat, repeat-x, repeat-y */
  
  /* Posición */
  background-position: center center;
  
  /* Tamaño */
  background-size: cover; /* contain, 6.25em, 50% */
  
  /* Fijación */
  background-attachment: fixed; /* scroll, local */
  
  /* Shorthand */
  background: url('img.jpg') no-repeat center / cover;
}

Gradientes

/* Gradiente lineal */
background: linear-gradient(
  to right,
  #ff0000,
  #00ff00
);

/* Con múltiples colores y paradas */
background: linear-gradient(
  45deg,
  red 0%,
  yellow 50%,
  green 100%
);

/* Gradiente radial */
background: radial-gradient(
  circle,
  #ff0000,
  #0000ff
);

/* Gradiente cónico */
background: conic-gradient(
  red,
  yellow,
  green,
  blue,
  red
);

Flexbox

Sistema de diseño unidimensional (fila o columna) para organizar elementos.

Ventajas: Alineación perfecta, distribución del espacio, orden flexible, ideal para componentes.

Contenedor Flex

.contenedor {
  display: flex;
  
  /* Dirección */
  flex-direction: row; /* row-reverse, column, column-reverse */
  
  /* Ajuste de línea */
  flex-wrap: wrap; /* nowrap, wrap-reverse */
  
  /* Justificación (eje principal) */
  justify-content: center;
  /* flex-start, flex-end, space-between, space-around, space-evenly */
  
  /* Alineación (eje cruzado) */
  align-items: center;
  /* flex-start, flex-end, stretch, baseline */
  
  /* Alineación de múltiples líneas */
  align-content: space-between;
  
  /* Espacio entre elementos */
  gap: 1.25em;
}

Elementos Flex

.item {
  /* Crecimiento */
  flex-grow: 1; /* proporción de crecimiento */
  
  /* Encogimiento */
  flex-shrink: 1;
  
  /* Tamaño base */
  flex-basis: 12.5em; /* auto, 0, 50% */
  
  /* Shorthand */
  flex: 1 1 12.5em; /* grow shrink basis */
  
  /* Auto-alineación */
  align-self: flex-end;
  
  /* Orden visual */
  order: 2; /* -1, 0, 1, 2... */
}
Tip: flex: 1 es equivalente a flex: 1 1 0 y hace que los elementos ocupen el mismo espacio.

CSS Grid

Sistema de diseño bidimensional (filas y columnas) más potente que Flexbox.

Contenedor Grid

.grid-container {
  display: grid;
  
  /* Definir columnas */
  grid-template-columns: 12.5em 1fr 2fr;
  /* También: repeat(3, 1fr), auto, minmax(6.25em, 1fr) */
  
  /* Definir filas */
  grid-template-rows: 6.25em auto 3.125em;
  
  /* Espaciado */
  gap: 1.25em; /* row-gap, column-gap */
  
  /* Áreas nombradas */
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  
  /* Alineación horizontal */
  justify-items: center; /* start, end, stretch */
  
  /* Alineación vertical */
  align-items: center;
  
  /* Alineación del grid completo */
  justify-content: center;
  align-content: center;
}

Elementos Grid

.grid-item {
  /* Posición por líneas */
  grid-column: 1 / 3; /* de línea 1 a 3 */
  grid-row: 2 / 4;
  
  /* Span (expandir) */
  grid-column: span 2; /* ocupar 2 columnas */
  grid-row: span 3;
  
  /* Shorthand */
  grid-area: 2 / 1 / 4 / 3; /* row-start / col-start / row-end / col-end */
  
  /* Por área nombrada */
  grid-area: header;
  
  /* Auto-alineación */
  justify-self: end;
  align-self: start;
}

Grid Responsive

/* Grid auto-responsive */
.grid {
  display: grid;
  grid-template-columns: repeat(
    auto-fill,
    minmax(15.625em, 1fr)
  );
  gap: 2rem;
}

/* auto-fill: crea tantas columnas como quepan */
/* auto-fit: similar pero colapsa columnas vacías */
/* minmax: tamaño mínimo y máximo */
Grid vs Flexbox:
  • Grid: layouts completos en 2D
  • Flexbox: componentes en 1D
  • ¡Ambos se complementan!

Diseño Responsive

Crear sitios que se adapten a diferentes dispositivos y tamaños de pantalla.

Media Queries

/* Móvil primero (mobile-first) */
.container {
  width: 100%;
  padding: 1rem;
}

/* Tablet */
@media (min-width: 48em) {
  .container {
    width: 46.875em;
    margin: 0 auto;
  }
}

/* Desktop */
@media (min-width: 64em) {
  .container {
    width: 62.5em;
  }
}

/* Desktop grande */
@media (min-width: 80em) {
  .container {
    width: 75em;
  }
}

Viewport Meta Tag

<!-- En el head del HTML -->
<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">

Unidades Responsive

/* Viewport Width/Height */
.hero {
  width: 100vw; /* 100% del ancho del viewport */
  height: 100vh; /* 100% del alto del viewport */
}

/* Porcentajes */
.sidebar {
  width: 25%; /* relativo al padre */
}

/* Clamp: valor fluido con límites */
.text {
  font-size: clamp(1rem, 2.5vw, 2rem);
  /* mínimo, preferido, máximo */
}

Imágenes Responsive

/* CSS */
img {
  max-width: 100%;
  height: auto;
  display: block;
}

/* HTML con srcset */
<img src="small.jpg"
     srcset="medium.jpg 768w,
             large.jpg 1024w"
     alt="Descripción">
Breakpoints comunes:
  • 320px - 480px: Móvil
  • 481px - 768px: Tablet
  • 769px - 1024px: Laptop
  • 1025px+: Desktop

Animaciones y Transiciones

Transiciones

Cambios suaves entre estados:

.button {
  background: blue;
  transform: scale(1);
  
  /* Transición */
  transition: all 0.3s ease;
  /* propiedad duración timing-function delay */
}

.button:hover {
  background: darkblue;
  transform: scale(1.1);
}

/* Propiedades específicas */
.box {
  transition-property: background-color, transform;
  transition-duration: 0.3s, 0.5s;
  transition-timing-function: ease, ease-in-out;
  transition-delay: 0s, 0.1s;
}

Keyframes

Animaciones más complejas:

/* Definir animación */
@keyframes deslizar {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(6.25em);
  }
}

/* Con porcentajes */
@keyframes rebote {
  0% { transform: translateY(0); }
  50% { transform: translateY(-3.125em); }
  100% { transform: translateY(0); }
}

/* Aplicar animación */
.elemento {
  animation: deslizar 2s ease-in-out infinite;
  /* nombre duración timing-function iteraciones */
}

/* Propiedades detalladas */
.animado {
  animation-name: rebote;
  animation-duration: 1s;
  animation-timing-function: ease;
  animation-delay: 0.5s;
  animation-iteration-count: infinite; /* o número */
  animation-direction: alternate; /* normal, reverse, alternate-reverse */
  animation-fill-mode: forwards; /* none, backwards, both */
  animation-play-state: running; /* paused */
}

Transforms

/* Traslación */
transform: translate(3.125em, 6.25em);
transform: translateX(3.125em);
transform: translateY(6.25em);

/* Rotación */
transform: rotate(45deg);
transform: rotateX(45deg);
transform: rotateY(45deg);

/* Escala */
transform: scale(1.5);
transform: scaleX(2);
transform: scaleY(0.5);

/* Sesgo */
transform: skew(10deg, 20deg);

/* Múltiples transformaciones */
transform: translate(3.125em, 3.125em) rotate(45deg) scale(1.5);
Timing functions:
  • ease: lento-rápido-lento
  • linear: velocidad constante
  • ease-in: acelera al inicio
  • ease-out: desacelera al final
  • cubic-bezier(): personalizado

Técnicas Avanzadas

Variables CSS

/* Definir variables globales */
:root {
  --color-primario: #3498db;
  --color-secundario: #2ecc71;
  --espaciado: 1rem;
  --fuente-principal: 'Poppins', sans-serif;
}

/* Usar variables */
.button {
  background: var(--color-primario);
  padding: var(--espaciado);
  font-family: var(--fuente-principal);
}

/* Con fallback */
.elemento {
  color: var(--color-personalizado, #000);
}

/* Variables locales */
.card {
  --card-padding: 2rem;
  padding: var(--card-padding);
}

Calc()

/* Cálculos dinámicos */
.sidebar {
  width: calc(100% - 18.75em);
}

.elemento {
  height: calc(100vh - 5em);
  margin: calc(2rem + 0.625em);
  font-size: calc(1rem + 1vw);
}

/* Con variables */
:root {
  --navbar-height: 3.75em;
}

.content {
  height: calc(100vh - var(--navbar-height));
}

Filtros

img {
  /* Desenfoque */
  filter: blur(0.3125em);
  
  /* Brillo */
  filter: brightness(150%);
  
  /* Contraste */
  filter: contrast(200%);
  
  /* Escala de grises */
  filter: grayscale(100%);
  
  /* Tono */
  filter: hue-rotate(90deg);
  
  /* Invertir */
  filter: invert(100%);
  
  /* Opacidad */
  filter: opacity(50%);
  
  /* Saturación */
  filter: saturate(200%);
  
  /* Sepia */
  filter: sepia(100%);
  
  /* Combinar filtros */
  filter: brightness(110%) contrast(120%) blur(0.125em);
}

Clip-path

/* Círculo */
.circulo {
  clip-path: circle(50%);
}

/* Elipse */
.elipse {
  clip-path: ellipse(50% 30%);
}

/* Polígono */
.triangulo {
  clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
}

/* Hexágono */
.hexagono {
  clip-path: polygon(
    50% 0%, 100% 25%, 100% 75%, 
    50% 100%, 0% 75%, 0% 25%
  );
}

Backdrop-filter

/* Efecto vidrio esmerilado */
.glassmorphism {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(0.625em);
  border: 0.0625em solid rgba(255, 255, 255, 0.2);
}

/* Modal con fondo difuminado */
.modal {
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(0.3125em) brightness(80%);
}
Compatibilidad: Algunas propiedades avanzadas pueden no funcionar en navegadores antiguos. Usa caniuse.com para verificar.
