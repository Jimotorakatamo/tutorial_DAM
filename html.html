Aprende HTML Moderno

HTML es el lenguaje fundamental de la web. Con él podrás crear la estructura básica de cualquier página web. En este tutorial, aprenderás desde los conceptos básicos hasta las características más modernas de HTML5.

Estructurado

Organiza el contenido de forma lógica

Accesible

Compatible con lectores de pantalla

Responsivo

Se adapta a todos los dispositivos

Estructura Básica

Todo documento HTML tiene una estructura fundamental que define su organización. Aquí tienes un ejemplo de la estructura básica:

DOCTYPE

Define el tipo de documento

HTML

Elemento raíz del documento

HEAD

Metadatos y recursos

BODY

Contenido visible

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Hola, Mundo</h1>
    <p>Este es mi primer documento HTML.</p>
</body>
</html>

¡A Practicar!

Crea tu propio archivo ejercicio.html y replica la estructura básica. Cambia el título y el contenido del párrafo por algo personal.

Etiquetas Esenciales

Las etiquetas HTML son los bloques básicos de construcción de una página web. Aquí tienes las más importantes:

Encabezados

h1 hasta h6

Párrafos

Texto y contenido

Imágenes

Contenido visual

Enlaces

Navegación

<!-- Encabezados -->
<h1>Título Principal</h1>
<h2>Subtítulo</h2>

<!-- Párrafos y texto -->
<p>Este es un párrafo con <strong>texto importante</strong> y <em>énfasis</em>.</p>

<!-- Enlaces -->
<a href="https://ejemplo.com">Visita mi sitio web</a>

<!-- Imágenes -->
<img src="imagen.jpg" alt="Descripción de la imagen">

<!-- Listas -->
<ul>
    <li>Elemento de lista no ordenada</li>
    <li>Otro elemento</li>
</ul>

HTML Semántico

El HTML semántico da significado a la estructura de tu página, mejorando la accesibilidad y el SEO.

Nav

Menú de navegación

Article

Contenido independiente

Section

Secciones temáticas

Footer

Pie de página

<header>
    <nav>
        <ul>
            <li><a href="#inicio">Inicio</a></li>
            <li><a href="#servicios">Servicios</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h1>Título del Artículo</h1>
        <section>
            <h2>Sección 1</h2>
            <p>Contenido de la sección...</p>
        </section>
    </article>
    
    <aside>
        <h3>Contenido Relacionado</h3>
        <p>Enlaces y recursos adicionales...</p>
    </aside>
</main>

<footer>
    <p>© 2025 Mi Sitio Web</p>
</footer>

Formato de Texto Avanzado

HTML ofrece una gran variedad de etiquetas para dar formato específico al texto, más allá de la negrita y la cursiva.

<p>Esto es <strong>importante</strong> y esto es <em>enfatizado</em>.</p>
<p>H<sub>2</sub>O y 2<sup>3</sup> = 8</p>
<p><mark>Texto resaltado</mark> para destacar.</p>

¡A Practicar!

En tu archivo ejercicio.html, añade un nuevo párrafo que contenga al menos una palabra en negrita, otra en cursiva, y una fórmula química o matemática usando o .

Listas

Multimedia en HTML

HTML5 facilitó enormemente la inserción de contenido multimedia sin necesidad de plugins externos.

<!-- Imagen con texto alternativo -->
<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">

<!-- Audio con controles -->
<audio controls src="ruta/a/tu/audio.mp3">Tu navegador no soporta audio.</audio>

<!-- Video con controles, ancho y alto -->
<video controls width="300" src="ruta/a/tu/video.mp4">Tu navegador no soporta video.</video>

¡A Practicar!

Busca una imagen que te guste en internet, copia su dirección y añádela a tu página con la etiqueta <img>. Asegúrate de ponerle un texto alternativo descriptivo.

Tablas

Las tablas se usan para mostrar datos tabulares de forma organizada. Es importante usar su estructura semántica.

<table>

    </thead>

</table>

¡A Practicar!

Crea una tabla sencilla con dos columnas: "Producto" y "Precio". Añade dos filas con datos de ejemplo, como "Portátil" y "1200€".

Metaetiquetas y SEO

Las metaetiquetas en el <head> proporcionan información crucial para los navegadores y los motores de búsqueda (SEO).

<head>
    <!-- Título de la página (muy importante para SEO) -->
    <title>Título Descriptivo de la Página</title>

    <!-- Descripción para los motores de búsqueda -->
    <meta name="description" content="Una descripción concisa de tu página (aprox. 155 caracteres).">

    <!-- Palabras clave (menos relevante hoy en día, pero no hace daño) -->
    <meta name="keywords" content="html, tutorial, web, desarrollo">

    <!-- Protocolo Open Graph para redes sociales (Facebook, LinkedIn...) -->
    <meta property="og:title" content="Título para Redes Sociales">
    <meta property="og:description" content="Descripción que aparecerá al compartir.">
    <meta property="og:image" content="https://ejemplo.com/imagen-preview.jpg">
    <meta property="og:url" content="https://ejemplo.com/pagina.html">
</head>

Atributos Avanzados

Además de los atributos comunes, HTML ofrece otros para funcionalidades más específicas.

Atributos data-*

Permiten almacenar información extra y personalizada en cualquier elemento HTML, que luego puede ser fácilmente accedida con JavaScript.

<div id="usuario" data-id="12345" data-role="admin">
    John Doe
</div>
<script>
    const usuarioDiv = document.getElementById('usuario');
    console.log(usuarioDiv.dataset.id); // "12345"
    console.log(usuarioDiv.dataset.role); // "admin"
</script>

Gráficos en la Web

HTML5 introdujo dos potentes formas de crear gráficos directamente en el navegador.

Canvas

Un lienzo para dibujar gráficos 2D y 3D dinámicamente con JavaScript. Ideal para animaciones, juegos y visualización de datos en tiempo real.

SVG (Scalable Vector Graphics)

Gráficos vectoriales basados en XML. Son escalables sin perder calidad y se pueden manipular con CSS y JavaScript. Perfectos para logos e iconos.

<!-- Un círculo simple con SVG -->
<svg width="100" height="100">
   <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

<!-- Un lienzo para dibujar con JavaScript -->
<canvas id="miCanvas" width="200" height="100" style="border:1px solid #000;"></canvas>

¡A Practicar!

Copia el código del círculo SVG en tu página. Intenta cambiar el color de relleno (fill) a "blue" y el radio (r) a "20".

APIs de HTML5

HTML5 no es solo etiquetas, también es un conjunto de APIs de JavaScript que dan superpoderes al navegador.

Web Storage

Permite almacenar datos en el navegador del usuario con localStorage (persistente) y sessionStorage (por sesión).

Geolocation

Permite al usuario compartir su ubicación con la aplicación web.

Web Workers

Ejecuta scripts en segundo plano sin congelar la interfaz de usuario.

// Ejemplo de uso de Local Storage
localStorage.setItem('nombreUsuario', 'Ana');
const usuario = localStorage.getItem('nombreUsuario');
console.log(usuario); // "Ana"

// Ejemplo de uso de Geolocation
navigator.geolocation.getCurrentPosition(pos => {
    console.log(`Latitud: ${pos.coords.latitude}, Longitud: ${pos.coords.longitude}`);
});

¡A Practicar!

Abre la consola de desarrollador de tu navegador (F12), ve a la pestaña "Consola" y pega el código de localStorage. Luego, recarga la página y comprueba en la pestaña "Aplicación" -> "Almacenamiento local" que el dato sigue ahí.
