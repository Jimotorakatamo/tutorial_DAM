switch

String dia = "Lunes";

switch (dia) {
    case "Lunes":
        System.out.println("Inicio de semana");
        break;
    case "Viernes":
        System.out.println("¡Por fin viernes!");

    case "Domingo":
        System.out.println("Fin de semana");

// Switch moderno (Java 14+)
String mensaje = switch (dia) {
    case "Lunes" -> "Inicio de semana";
    case "Viernes" -> "¡Por fin viernes!";
    case "Sábado", "Domingo" -> "Fin de semana";
    default -> "Día normal";
};

Clases y Objetos

public class Persona {
    // Atributos
    private String nombre;
    private int edad;
    
    // Constructor
    public Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    
    // Getters y Setters
    public String getNombre() {
        return nombre;
    }
    
    public void setNombre(String nombre) {
        this.nombre = nombre;
    }
    
    // Método
    public void saludar() {
        System.out.println("Hola, soy " + nombre);
    }
}

// Uso
Persona persona = new Persona("Ana", 25);
persona.saludar(); // "Hola, soy Ana"

Herencia

    private final int y;

        this.x = x;
        this.y = y;

Streams (Java 8+)

import java.util.List;
import java.util.stream.Collectors;

List<Integer> numeros = List.of(1, 2, 3, 4, 5, 6);

// Filtrar pares
List<Integer> pares = numeros.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());
// [2, 4, 6]

// Mapear (duplicar cada número)
List<Integer> dobles = numeros.stream()
    .map(n -> n * 2)
    .collect(Collectors.toList());
// [2, 4, 6, 8, 10, 12]

// Sumar todos
int suma = numeros.stream()
    .reduce(0, Integer::sum);
// 21

Manejo de Excepciones

try-catch

try {
    int resultado = 10 / 0; // Error: división por cero
} catch (ArithmeticException e) {
    System.out.println("Error: " + e.getMessage());
} finally {
    System.out.println("Esto siempre se ejecuta");
}

Múltiples excepciones

} catch (Exception e) {
    System.out.println("Otro error: " + e);
}

// Java 7+ (Multi-catch)
try {
    // código que puede fallar

}

Lanzar excepciones

public void validarEdad(int edad) {
    if (edad < 0) {
        throw new IllegalArgumentException(
            "La edad no puede ser negativa"
        );
    }
    if (edad < 18) {
        throw new IllegalArgumentException(
            "Debe ser mayor de edad"
        );
    }
}

// Uso
try {
    validarEdad(-5);
} catch (IllegalArgumentException e) {
    System.out.println(e.getMessage());
}
Buena práctica: Captura excepciones específicas, no solo Exception genérico. Usa finally para limpiar recursos.

Streams API

Los streams en Java son secuencias de datos que permiten procesar colecciones de manera funcional. Se introdujeron en Java 8 y son parte del paquete java.util.stream.

filter()

Filtra elementos según una condición

map()

Transforma cada elemento

reduce()

Combina elementos en un resultado

collect()

Recolecta en una colección

import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<String> nombres = Arrays.asList("Ana", "Luis", "Alberto", "María");

// Filtrar nombres que empiecen con "A"
nombres.stream()
       .filter(nombre -> nombre.startsWith("A"))
       .forEach(System.out::println); // Ana, Alberto

// Transformar a mayúsculas
List<String> mayusculas = nombres.stream()
       .map(String::toUpperCase)
       .collect(Collectors.toList());

// Contar elementos
long count = nombres.stream()
       .filter(n -> n.length() > 4)
       .count(); // 3

Expresiones Lambda

Las expresiones lambda son una forma concisa de representar funciones anónimas en Java. Se utilizan principalmente para implementar interfaces funcionales.

Sintaxis básica

(parámetros) -> expresión

(parámetros) -> { código; }

// Interfaz funcional
interface Operacion {
    int calcular(int a, int b);
}

// Usando una expresión lambda
Operacion suma = (a, b) -> a + b;
Operacion resta = (a, b) -> a - b;

int resultado1 = suma.calcular(5, 3); // 8
int resultado2 = resta.calcular(10, 4); // 6

// Lambdas con Runnable
Runnable tarea = () -> {
    System.out.println("Ejecutando tarea");
};
new Thread(tarea).start();

// Lambdas con Comparator
List<String> nombres = Arrays.asList("Carlos", "Ana", "Beatriz");
nombres.sort((a, b) -> a.compareTo(b));

Referencias a métodos

// Equivalente a: nombre -> System.out.println(nombre)
nombres.forEach(System.out::println);

// Equivalente a: s -> s.toUpperCase()
nombres.stream().map(String::toUpperCase);

// Equivalente a: () -> new ArrayList<>()
Supplier<List<String>> listSupplier = ArrayList::new;

¡A Practicar!

Crea una expresión lambda que tome un String y devuelva su longitud. Puedes usar la interfaz funcional Function<String, Integer>.

// Sealed classes: jerarquías controladas
public sealed interface Pedido 
    permits PedidoOnline, PedidoPresencial {}

public record PedidoOnline(String id, double total) 
    implements Pedido {}

public final class PedidoPresencial implements Pedido {
    private final String id;
    private final double total;
    
    public PedidoPresencial(String id, double total) {
        this.id = id;
        this.total = total;
    }
}

Pattern Matching

static String describir(Pedido pedido) {
    return switch (pedido) {
        case PedidoOnline online -> 
            "Pedido web #" + online.id() + " - " + online.total() + "€";
        case PedidoPresencial presencial -> 
            "Pedido tienda #" + presencial.id();
    };
}

if (obj instanceof String s && s.length() > 5) {

Programación Funcional

Java incorpora conceptos funcionales mediante Streams, Optional, CompletableFuture y librerías como Project Reactor.

Interfaces funcionales principales

  • Predicate<T> - Condición que devuelve boolean
  • Function<T,R> - Transforma T en R
  • Consumer<T> - Consume T sin devolver nada
  • Supplier<T> - Produce un T
List<String> modulos = List.of("Programación", "Marcas", "Entornos");

// Pipeline funcional
String resumen = modulos.stream()
    .map(String::toUpperCase)
    .filter(n -> n.contains("O"))
    .sorted()
    .collect(Collectors.joining(", "));

// Optional para evitar nulls
Optional<String> primero = modulos.stream()
    .filter(m -> m.startsWith("P"))
    .findFirst();

primero.ifPresent(System.out::println);

// CompletableFuture para asincronía
CompletableFuture<String> saludo = 
    CompletableFuture.supplyAsync(() -> "Hola DAM")
        .thenApply(String::toUpperCase);

saludo.thenAccept(System.out::println);

import javax.persistence.*;

@Entity
@Table(name = "alumnos")
public class Alumno {
    @Id 
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    private String nombre;
    private String ciclo;
    private double notaMedia;
    
    // Constructores, getters y setters
}

// Spring Data Repository
public interface AlumnoRepository 
    extends JpaRepository<Alumno, Long> {
    
    List<Alumno> findByCiclo(String ciclo);
    List<Alumno> findByNotaMediaGreaterThan(double nota);
}

// Servicio
@Service
public class AlumnoService {
    private final AlumnoRepository repository;
    
    public AlumnoService(AlumnoRepository repository) {
        this.repository = repository;
    }
    
    public List<Alumno> obtenerAprobados() {
        return repository.findByNotaMediaGreaterThan(5.0);
    }
}
